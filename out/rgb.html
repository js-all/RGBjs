 La tansparance de la couleur, valeur entre 0 et 1. Si non préciser prend la valeur defini dans rgb.config.defaultAlpha.

renvoi la couleur sou forme de string. exemple: new rgb(0, 255, 0, 1).get() renvoi "rgba(0, 255, 0, 1)"
Parameters:
Name Type Attributes Default Description
alpha boolean <optional>
 true precise si la transparence de la couleure doir etre donne. example new rgb(0, 255, 0, 1).get() renvoie "rgba(0, 255, 0, 1)" alors que new rgb(0, 255, 0, 1).get(false) renvoie "rgb(0, 255, 0)", si non preciser prend true.
